# SunHub Design System

Energy operations control tower for Techos Rentables (200+ solar plants, multi-brand: Growatt, Huawei, DeyeCloud, Hoymiles, SRNE).

## Brand feel
- Clean, data-dense, serious but optimistic (solar/sustainability).
- Primary green #16A34A (growth, energy ok).
- Accent yellow #FACC15 (sun/alert-warn).
- Info blue #0EA5E9 (data).
- Danger red #DC2626 (fault/offline). Success green #16A34A. Warning amber #F59E0B.

## Layout
- Desktop-first SaaS dashboard with persistent left sidebar (icons + labels) and top header with search + user menu.
- Cards with 12px radius, subtle shadow, white background on light gray canvas (#F8FAFC).
- Data-dense: small metric cards, sparkline charts, status pills.

## Components
- Status pills: Online (green), Warning (amber), Offline (red), Syncing (blue).
- Brand badges: small logos/chips for Growatt, Huawei, DeyeCloud, Hoymiles, SRNE next to devices.
- KPI cards: big number + delta vs prior period + sparkline.
- Tables: zebra off, sticky header, filter chips above.
- Charts: line charts for generation, bar for daily kWh, gauges for PR/uptime.

## Tone
- Spanish (Colombia) copy. Technical but approachable.
- Use ⚡ energy icon sparingly for accent.
